# AI-Generated Products & Pages

## Scale Your Store With AI Content

One product is a start. But real stores need multiple products, compelling descriptions, and pages that rank on Google. AI does 90% of this work.

## Product Descriptions That Sell

For every product, use this prompt:

```
Write a product description for my Australian e-commerce store:

Product: [NAME]
Price: $[PRICE] AUD
Target customer: [WHO]
Key benefit: [MAIN REASON TO BUY]

Write:
1. A punchy headline (under 10 words)
2. A 2-paragraph description (conversational, not salesy)
3. 5 bullet point features
4. A short "who it's for" section
5. Tone: Australian, direct, no corporate speak

Optimise for SEO keyword: [MAIN KEYWORD]
```

**Pro tip:** Run this for every product, then read each one out loud. If it sounds like a robot wrote it, edit the awkward parts. AI gets 80% right — your 20% edit makes it human.

## Batch-Generate Products

If you have 10+ products to list, use this batch approach:

```
I have these products to list on my store. For each one, write:
- Headline, description, 5 features, SEO keyword

Products:
1. [Product A] — $[Price] AUD — [one-line description]
2. [Product B] — $[Price] AUD — [one-line description]
3. [Product C] — $[Price] AUD — [one-line description]
[continue for all products]

Tone: conversational Australian English. Each description should feel unique — don't use the same sentence structure for every product.
```

## Essential Pages Every Store Needs

Beyond products, you need these pages. AI writes them all:

### About Page
```
Write an About page for my store [STORE NAME]. I'm a [your background] based in [city, Australia]. The store sells [products] because [why you started]. Tone: genuine, personal, not corporate. Include: my story (2 paragraphs), what makes us different (3 bullets), and a call to action to browse products.
```

### FAQ Page
```
Write an FAQ page for my Australian e-commerce store selling [products]. Include 8-10 questions covering: shipping times, returns policy, payment methods, sizing/specs, contact info. Base shipping on Australia Post standard rates. Returns within 30 days. We accept Visa, Mastercard, Amex, Apple Pay, Google Pay.
```

### Shipping & Returns
```
Write a Shipping & Returns policy for an Australian online store:
- Standard shipping: $9.95 AUD flat rate or free over $75
- Express shipping: $14.95 AUD (1-3 business days)
- We ship with Australia Post and Sendle
- Returns accepted within 30 days, item must be unused
- Refunds processed within 5 business days
- Customer pays return shipping unless item is faulty
Make it clear, fair, and compliant with Australian Consumer Law.
```

### Contact Page
Simple — just your email, a contact form, and estimated response time. AI can build the form.

## SEO Product Pages

Every product page should target a keyword. Tell AI:

```
Optimise my product page for the keyword "[YOUR KEYWORD]":
- Write a meta title (under 60 characters, include keyword)
- Write a meta description (under 160 characters, include keyword, compelling)
- The product description should naturally include the keyword 2-3 times
- Add alt text for the product image that includes the keyword
```

## Product Photography (Without a Camera)

For **digital products:**
- Use Canva mockup templates (free) — put your product on a laptop/phone screen
- Screenshot your actual product and frame it nicely

For **physical products:**
- Use your supplier's photos (most dropship suppliers provide them)
- For print-on-demand: use the mockup generator from Printful/Printify
- If you have the actual product: use your phone + natural lighting near a window. White background. Simple.

**AI image generation** (Midjourney, DALL-E) works for lifestyle/marketing images but NOT for product photos — customers need to see the real thing.
